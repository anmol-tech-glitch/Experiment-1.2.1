# ✍️ Live Character Counter (React)

A simple React app that counts characters in real-time as the user types.

---

## 🚀 Features
- Real-time character counting
- Character limit with warning
- Responsive UI
- Smooth CSS transitions

---

## 🛠️ Tech Stack
- React.js (useState)
- CSS3

---

## ▶️ How to Run
1. Create React app:
   npx create-react-app counter-app
2. Replace App.js and App.css
3. Run:
   npm start
